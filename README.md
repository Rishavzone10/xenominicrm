## 📌 Overview
The **Mini CRM Dashboard** is a full-stack marketing automation tool designed to streamline customer management. It empowers businesses to create targeted campaigns, segment audiences dynamically, and track the delivery performance of personalized communication.

---

## ✨ Key Features

* **📊 Dashboard Analytics:** Real-time visualization of customer data, campaign status, and delivery success rates.
* **🎯 Audience Segmentation:** Build custom target groups using flexible AND/OR logic.
* **📣 Campaign Management:** Create and track multi-channel marketing campaigns with ease.
* **💬 Personalized Messaging:** Automated system for personalized delivery (e.g., *Hi [Name]!*).
* **⚡ Asynchronous Processing:** Efficient handling of large message batches using an in-memory queue system.
* **🔐 Secure Auth:** Integrated Google OAuth 2.0 for secure, persistent user access.
